# ICTK Machine Learning with Generative AI

## Course Overview

This repository contains learning materials, notebooks, datasets,
assignments, and project templates for the 5-Day ICTK Machine Learning
with Generative AI Training Program.

---

## Course Schedule

### Day 1
Python & Data Foundations for Machine Learning

- NumPy
- Pandas
- Data Cleaning
- Data Visualization
- Matplotlib
- Seaborn

### Day 2
Supervised Machine Learning

- ML Workflow
- Regression
- Classification
- Model Validation
- Evaluation Metrics

### Day 3
Model Optimization & Unsupervised Learning

- Feature Engineering
- Overfitting & Underfitting
- Hyperparameter Tuning
- Cross Validation
- Clustering

### Day 4
Generative AI & LLMs

- Introduction to GenAI
- LLM Concepts
- Prompt Engineering
- Zero-shot Prompting
- Few-shot Prompting
- API Usage
- Responsible AI

### Day 5
Integrating ML with GenAI

- Explainable AI
- SHAP
- Automated Insights
- Streamlit
- Mini Project
